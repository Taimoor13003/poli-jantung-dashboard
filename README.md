# Looker Next

Simple Next.js app that embeds a Google Looker Studio report on the homepage.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open http://localhost:3000

## Change report URL

Edit `app/page.js` and update `REPORT_URL`.

If the report fails to load, ensure the Looker Studio report is shared for public viewing and embedding.
